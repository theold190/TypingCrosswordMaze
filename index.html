    • The CrosswordMaze game

    A typing game where you navigate in a CrosswordMaze (using Crafty JS game engine)

    Project maintained by theold190 Hosted on GitHub Pages — Theme by mattgraham

    Crossword Maze is a game that was created at a summer DreamCode2012 event and prepared for Liberation Pixel Cup competition. It was created during 4-5 days with use of Crafty javascript game engine. The idea of the game is to practice typing while having fun. Go on with the instruction, and try to get a highest score!

    During the game you will need to complete 3 different stages. On every stage you need to get as many points as possible (by collecting golden an quest letters). To finish the stage you will need to go to the portal/hole in the ground/grab a bag of gold. Between the stages you will see a screen with a current game status.

    Here are some details.

    A main hero. Hero can move up, down, left, right in a maze. To move in a desired cell you need to type a letter that is located in that cell. If you mistype - your final score will be decreased.

    To complete a playing stage you will need to move a hero to the portal/hole/bag of gold. To do that position your hero next to the portal and press SPACE/ENTER.

    This is a starting screen. It shows you the current game stage (0 out of 3). There is also a task/quest in a game. To complete it you need to collect letters you see on the info screen. Each of them increases your score. If you collect all of them - you will get some extra score. Press SPACE/ENTER to proceed.

    This is a screen with the final game score. To restart the game press SPACE/ENTER.

    How to improve your score:

    • Collect more gold (yellow letters)
    • Avoid danger (red letters)
    • Avoid typing mistakes
    • Try to complete the quest
